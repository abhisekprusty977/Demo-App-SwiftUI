SwiftUIPro:
SwiftUIPro is a production-grade SwiftUI application built step-by-step in public.

Goals:
Demonstrate senior-level SwiftUI architecture and best practices
Build features incrementally using real-world workflows
Emphasize performance, concurrency, and long-term scalability

Tech Stack:
SwiftUI
MVVM architecture
async/await concurrency
Actors for thread safety
State-driven navigation
Each week introduces a new real-world concept, applied directly within the app.
